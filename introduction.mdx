## What is foreman-builder?

`foreman-builder` is a command-line tool for macOS that automates the creation of containerized [Foreman](https://theforeman.org/) development environments. It uses [OrbStack](https://orbstack.dev/) to run Rocky Linux 9 containers that feel as close to a local environment as possible — with localhost port forwarding, SSH access, and a fully configured Katello/Foreman stack.

<Note>
  foreman-builder currently only supports **Apple Silicon (arm64)** Macs. Intel-based Macs are not supported.
</Note>

## Why foreman-builder?

Setting up a Foreman development environment from scratch involves installing Ruby, Node.js, Ansible, configuring repositories, running playbooks, and wiring everything together. `foreman-builder` automates all of this so you can focus on development rather than setup.

## How it works

1. You run `foreman-builder create` in your terminal.
2. foreman-builder reads your optional `config.yml` for any extra packages to install.
3. It generates a cloud-init configuration using your username and package list.
4. It calls `orb create` to spin up a Rocky Linux 9 container on OrbStack.
5. Cloud-init bootstraps the container: installs dependencies, sets up Ruby/rbenv, clones [forklift](https://github.com/theforeman/forklift), and runs the Katello development playbook.
6. Once complete, you can SSH into the container and start developing.

## Requirements

- macOS with Apple Silicon (arm64)
- [Go](https://go.dev/doc/install) (for installation via `go install`)
- [OrbStack](https://orbstack.dev/) installed and running
